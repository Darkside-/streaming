## Ankhbot - Overlay for Bets

Version: 1.0.1

If you add this per BrowserSource use following sizes:
Width: 	700
Height: 500

Two variables to adjust if you want in the "client.js":

Line 16 "var showBets":
	"true" = If you want to display everyone who did Bet
	"false"= If you don't want to display those users

Line 16 "var maxAmountOfEntries":
	X      = Set that to any amount of people you want to display below the Giveaway Title (Remember 2 Column Layout)
